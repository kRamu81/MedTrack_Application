# Security Policy

## Supported Versions

Currently, the following versions of MedTrack are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them to the repository admin privately via our Discord server or directly via direct message to the admin. 

You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible depending on complexity.
